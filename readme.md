# Generate Mascot Search Links

This script accesses the Mascot search log. It then saves links to the searches that were carried out on your profile. The links are saved on an excel spreadsheet.

## Getting Started

### Dependencies

* [R](https://cran.r-project.org/bin/windows/base/)

### Installing

* Install R (version 4.5.2) and Rstudio
* Add the location of Rscript.exe to config.txt. If there are any spaces in the location, put " around the location.

### Executing program

* Double click on get-mascot-links.bat
* Enter username and password to login to the Mascot server.
* The Excel file will be saved to the script directory.

## Help

Note: the script can only access the last 50 searches on the Mascot server.
